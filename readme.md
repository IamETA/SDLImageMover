# Image Movement using SDL window library (x86)

 - Clone repostiry
 - Open in Visual Studio
 - Change linker location & include location under project options to your SDL\lib\x86\sdl2.lib and SDL\include
 - Compile & Run

 # External dependencies
  - SDL2.dll
  - SDL2_ttf.dll
  - FreeSans.ttf
  - zlib1.dll
